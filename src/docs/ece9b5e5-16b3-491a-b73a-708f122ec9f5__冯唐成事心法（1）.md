# 全书的结构
全书为四章：
**用好自己的天赋**
**人人都该懂战略**
**成事的自我修养**
**知可为，知不可为**

这本书整体来看，就好像是作者在四个维度下，探讨了一件件事儿，形形色色，包罗万象。有一说一，标题起的是真好，很多都是一眼就被抓住，那么我就挑出这些“一眼抓”的事，来进行理解沉淀。
